About Schedule Contacts

1on1

1on1 is a web application that allows you to schedule 1 on 1 meetings with your friends and colleagues.

Features

  • Share calendars
  • Set time preferences
  • Add contacts
  • Connect other calendars (Google Calendar, Apple Calendar)

It is built using the following technologies:

  • HTML
  • CSS
